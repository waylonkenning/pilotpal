# PPL Quest NZ

A gamified web application to help aspiring pilots navigate the journey to obtaining their Private Pilot License (PPL) in New Zealand.

## Tech Stack

- **Vue 3** - Modern reactive framework
- **TypeScript** - Type safety and better developer experience
- **Vite** - Fast development and build tooling
- **Custom CSS** - Clean, consistent design system
