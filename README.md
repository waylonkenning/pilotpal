# PPL Quest NZ

A gamified web application to help aspiring pilots navigate the journey to obtaining their Private Pilot License (PPL) in New Zealand.

## Tech Stack

- **Vue 3** - Modern reactive framework
- **TypeScript** - Type safety and better developer experience
- **Vite** - Fast development and build tooling
- **Custom CSS** - Clean, consistent design system

## Features

- 🎮 **Gamified Progress Tracking** - Turn your PPL journey into an engaging game
- 📊 **Flight Hours Logging** - Track dual, solo, cross-country hours
- 🏆 **Achievement System** - Unlock badges and milestones
- 📚 **Theory Progress** - Track exam preparation and completion
- 💰 **Cost Tracking** - Monitor training expenses
- 🗺️ **NZ-Specific** - Tailored for New Zealand aviation requirements

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```

This project follows Vue 3 Composition API best practices with TypeScript for type safety and a mobile-first CSS approach for responsive design.
